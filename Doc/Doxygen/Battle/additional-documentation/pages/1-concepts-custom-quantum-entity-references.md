## Custom %Quantum Entity References {#page-concepts-custom-quantum-entity-references-custom-references}

The @ref BattleCustomEntityReferences.qtn includes custom **%Quantum Entity** reference structs that act as wrappers for specific types of **%Quantum Entities**. <br/>
The @ref BattleCustomEntityReferences.cs includes extension methods for these custom **%Quantum Entity** structs. <br/>
The point of these files is to separate specific types of **%Quantum Entity** references and be able to add extension methods to these references. <br/>
