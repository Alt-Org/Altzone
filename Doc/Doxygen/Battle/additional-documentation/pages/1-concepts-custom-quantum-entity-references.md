## Custom %Quantum Entity References {#page-concepts-custom-quantum-entity-references-custom-references}

The @ref BattleCustomEntityReferences.qtn includes custom **%Quantum Entity** reference structs that act as wrappers for specific types of **%Quantum Entities**.  
The @ref BattleCustomEntityReferences.cs includes extension methods for these custom **%Quantum Entity** structs.  
The point of these files is to separate specific types of **%Quantum Entity** references and be able to add **Entity** specific extension methods to these references.
