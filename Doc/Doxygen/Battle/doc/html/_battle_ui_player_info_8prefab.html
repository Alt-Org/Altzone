Detailed Description

Prefab for Battle's PlayerInfo elements (PlayerName and CharacterButtons).

Used in both Battle's UI and BattleUiEditor in MenuUi. Displays player's characters, their health, and shield health.
Has a BattleUiMultiOrientationElement script which adds multi orientation functionality for the movable element.
