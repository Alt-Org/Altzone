Detailed Description

ProjectileSpawner Quantum System游못 [Systems]
Handles spawning the projectile at the beginning of the game.

An invisible entity with BattleProjectileSpawnerQComponent is created when system is initiated, which is then used to spawn the projectile when BattleGameState is changed to "Playing".
