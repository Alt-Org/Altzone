Detailed Description

Contains all battle events.

Events

Game Flow events - View Pre-Activate

  • synced event BattleViewWaitForPlayers {}
    Synced event. Used when waiting for for all players to get registered.
  • synced event BattleViewPlayerConnected {}
    Synced event. Used when a player connects to the game.
  • synced event BattleViewAllPlayersConnected {}
    Synced event. Used when all players have connected to the game.
  • synced event BattleViewInit {}
    Synced event. Used for initialization after players are ready.

Game Flow events

  • synced event BattleViewActivate {}
    Synced event. Used to activate view after initialization is ready.
  • synced event BattleViewGetReadyToPlay {}
    Synced event. Used when the start countdown is finished.
    .
  • synced event BattleViewGameStart {}
    Synced event. Used when the game starts.
  • synced event BattleViewGameOver { BattleTeamNumber WinningTeam; FP GameLengthSec }
    Synced event. Used when the game ends.

ViewInit events

  • synced event BattlePlayerViewInit { EntityRef Entity; FP ModelScale }
    Synced event. Used to initiate player's view model on Unity side.
  • synced event BattleSoulWallViewInit { EntityRef Entity; FP ModelScale; int EmotionIndicatorColorIndex; int DebugColorIndex }
    Synced event. Used to initiate SoulWalls's view models on Unity side.
  • synced event BattleStoneCharacterPieceViewInit { int WallNumber; BattleTeamNumber Team; int EmotionIndicatorColorIndex }
    Synced event. Used to set EmotionIndicatorColor for StoneCharacterPieces.

UI control events

  • synced event BattleViewSetRotationJoystickVisibility {bool IsVisible, BattlePlayerSlot Slot }
    Synced event. Used to set the rotation joystick UI element to be shown or hidden.

Gameplay events

  • synced event BattleChangeEmotionState { BattleEmotionState Emotion }
    Synced event. Used to change sprites etc. during game depending on given BattleEmotionState.
  • synced event BattleProjectileChangeSpeed { FP NewSpeed }
    Synced event. Used to update the projectile's trail based on it's speed.
  • synced event BattleProjectileChangeGlowStrength { FP Strength }
    Synced event. Used to change glow strength during game depending on the projectile's attack damage.
  • event BattlePlaySoundFX { BattleSoundFX Effect }
    Non-synced Event. Used to play sound effects during battle.
  • synced event BattleLastRowWallDestroyed { int WallNumber; BattleTeamNumber Team; FP LightrayRotation; BattleLightrayColor LightrayColor; BattleLightraySize LightraySize }
    Synced event. Used when a last row SoulWall segment is destroyed.
  • synced event BattleDiamondLanded { EntityRef Entity }
    Synced event. Used when a diamond stops traveling across the arena.
  • synced event BattleCharacterTakeDamage { EntityRef Entity; BattleTeamNumber Team; BattlePlayerSlot Slot; int CharacterNumber; FP HealthPercentage }
    Synced event. Used when a character takes damage.
  • synced event BattleShieldTakeDamage { EntityRef Entity; BattleTeamNumber Team; BattlePlayerSlot Slot; int CharacterNumber; FP DefenceValue }
    Synced event. Used when character's shield takes damage.
  • synced event BattleGiveUpStateChange {BattleTeamNumber Team; BattlePlayerSlot Slot; BattleGiveUpStateChange StateUpdate }
    Synced event. Used when player changes give up state.
