Detailed Description

Joystick UI Component (Unity MonoBehaviour script游못).
Handles Battle Ui joystick drag input functionality using IPointerDownHandler, IDragHandler and IPointerUpHandler interfaces.

This script is meant to be attached to BattleUiJoystick prefab's top level GameObject, since it calculates _joystickRadius from the same GameObject游못's RectTransform游못 component.
