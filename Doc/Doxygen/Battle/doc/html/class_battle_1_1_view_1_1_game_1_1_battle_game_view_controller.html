Detailed Description

Game view QuantumCallbacks class游못.
Initializes Battle UI elements, and controls their visibility and functionality.

Handles any functionality needed by the [UI Handlers] which need to access other parts of Battle, for example triggering the selection of another character. Accesses all of the UIHandlerReferences through the BattleUiController reference variable _uiController.
