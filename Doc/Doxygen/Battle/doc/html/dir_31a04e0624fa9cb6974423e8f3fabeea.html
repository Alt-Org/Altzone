Detailed Description

Contains documentation files for code generated by Quantum.
See [Documenting Quantum generated code].

Documentation can not be written in source because the code is generated, so it is written here to not have additional files in the Unity's Assets/ folder especially because Unity would create .meta files to keep track of the files for no reason.
