Detailed Description

DebugConsole UI Handler (Unity MonoBehaviour script游못).
Handles displaying the custom on-screen debug console.

Controlled by BattleDebugLogger

Shows only certain amount of log entries. First line shows the type of log and it's source and second line shows the message of the log.
Logs appear on console when BattleDebugLogger requests log to be shown.
Each log entry has a lifetime after which they disappear.
See SerializeFields for all settings.

When visible, automatically shows entry logs if there are any, otherwise debug console will be hidden.
When not visible, console is never shown.
See SetShow
