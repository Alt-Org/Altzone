Detailed Description

Class for defining bot behavior and other spec settings.

This class is used to define the data asset's structure, the data itself is not contained here.
Can be used to make multiple BattlePlayerBotQSpec data assets.
Filled with data from BattlePlayerBotQSpec data asset.
