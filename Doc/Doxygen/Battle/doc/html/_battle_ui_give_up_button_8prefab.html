Detailed Description

Prefab for Battle's GiveUp-button.

Used in both Battle's UI and BattleUiEditor in MenuUi. A button the local player can press to give up the current match.
Has a BattleUiMovableElement script since this BattleUi element doesn't need additional functionality.
