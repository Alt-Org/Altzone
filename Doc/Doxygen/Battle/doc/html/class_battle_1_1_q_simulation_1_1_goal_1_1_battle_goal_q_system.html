Detailed Description

Goal Quantum SystemSignalsOnly游못 [Systems]
Triggers the end of the game when the projectile hits a goal.

This system is activated by BattleCollisionQSystem when projectile hits a goal. System then changes BattleGameState to "GameOver" to initiate game over procedures.
