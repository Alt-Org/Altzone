Detailed Description

Text scaler Unity MonoBehaviour script游못.
Scales TextMeshProUGUI游못 text based on text box size and font size target percentage. Uses arrays to apply same settings to multiple texts.

Note: Disables font auto sizing and text wrapping for TextMeshProUGUI游못.

◆ ScaleTextBox()

void Altzone.Scripts.BattleUiShared.TextScaler.ScaleTextBox ( int index)
inlineprivate

Handles text box scaling while keeping aspect ratio with compatibility for widescreen.

Parameters
indexThe text index which text box to scale.

The way this works is that the text box will scale y/height anchors while keeping the _textBoxAspectRatio, and x/width anchors say the same.
If _holderRectTransform's current aspect ratio is wider than _startScalingHolderAspectRatio then the text box won't scale in height so that the text won't become too large.
