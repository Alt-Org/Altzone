Detailed Description

PlayerManager handles player management, allowing other classes to focus on gameplay logic.
Provides static methods to initialize, spawn, despawn, and query player-related data.

{Player Overview}
{Player Simulation Code Overview}

Handles initializing players that are present in the game, as well as spawning and despawning player characters.
Also contains the public and private PlayerHandle structs.
