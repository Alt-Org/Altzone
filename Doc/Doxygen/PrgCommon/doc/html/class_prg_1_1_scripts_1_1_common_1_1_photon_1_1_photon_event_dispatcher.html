Detailed Description

Wrapper and helper for PhotonNetwork.RaiseEvent for sending and receiving events on one place.

Note 1. We use lazy initialization and can not survive level load.
Note 2. By default we use PhotonNetwork.NetworkingClient.LoadBalancingPeer.ReuseEventInstance set to true. Call DisableReuseEventInstance() if you do not want this behaviour.
