Detailed Description

Static helper class to handle basic PhotonNetwork operations in convenient and consistent way.
https://doc-api.photonengine.com/en/pun/v2/class_photon_1_1_pun_1_1_photon_network.html.

Note (1) that by default we use

PhotonNetwork.AutomaticallySyncScene = true

.
Note (2) that PhotonNetwork OfflineMode must be handled separately if needed.
