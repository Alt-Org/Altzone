Detailed Description

Simple 'localhost' HTTP Server based on HttpListener.
You can send requests to following URLs when using port number 8090:
http://localhost:8090/ or http://127.0.0.1:8090/ and http://&lt;COMPUTERNAME&gt;:8090/.
