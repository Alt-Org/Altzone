Detailed Description

Convenience class for platform detection to access platform specific features.
Note that we have distinct separation of IsEditor and IsDevelopmentBuild, UNITY considers that they are "same".

Most of these are getters because static code analysis will otherwise complain about using compile time constants that are always true or false.
