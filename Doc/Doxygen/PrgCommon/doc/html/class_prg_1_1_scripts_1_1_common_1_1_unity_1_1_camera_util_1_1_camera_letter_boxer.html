Detailed Description

Preserve your 2D game's aspect ratio adding letterbox / pillarbox (bars) if required. See: https://github.com/rabidgremlin/LetterBoxer.

This could be used instead of CameraAspectRatio. They are implemented differently and I do not know which one is better or more suitable for the task.
